# Community Presets

Community presets live in this directory.

To contribute one:

1. Copy `template.json`
2. Rename it to a slug-like filename such as `my-whoop-preset.json`
3. Fill in the metadata and preset values
4. Add the file path to `../index.json`
5. Open a pull request

If the file is valid, the website will load it automatically and show the preset name and author in the voice browser.
